## Pedagogical Design

These diagrams are designed to teach solving techniques progressively:

### Diagram 1: Reference Table
- Clean, scannable table format
- Green highlighting for unique combinations (memorize these!)
- Shows both 2-cell and 3-cell runs
- Includes number of options for each sum

### Diagram 2: Elimination Walkthrough (Two-Page Layout)
- **Page 1:** Introduction and ACROSS run analysis (Grids 1-2)
- **Page 2:** DOWN run analysis, intersection finding, and solution (Grids 3-5)
- Step-by-step process with annotations across both pages
- Color-coded intersecting runs
- Shows listing possibilities, checking intersections, and reaching solution
- Builds confidence in the elimination method through progressive disclosure

### Diagram 3: Cascade Effect
- Before/after comparison layout
- Demonstrates the "aha!" moment when one digit unlocks many
- Encourages learners to look for cascade opportunities

### Diagram 4: Starting Strategies
- Visual legend with icons (stars for corners, circles for constraints)
- Multiple color highlights show different strategic entry points
- Teaches where to begin on an empty grid

### Diagram 5: Advanced Preview
- Shows pencil mark notation
- Demonstrates multi-run analysis
- Prepares learners for intermediate-level complexity

### Diagram 6: Error Recovery
- Side-by-side comparison of wrong vs. corrected puzzle
- Detailed annotations explain how to trace the error
- Reduces frustration by teaching troubleshooting

## Color Coding System

Chapter 2 uses an educational color scheme:

- **Light Blue** - Indicates cells in one run (for teaching)
- **Light Yellow** - Indicates cells in another run (for teaching)
- **Light Green** - Shows unique combinations or correct answers
- **Green** - Solved/correct cells
- **Yellow/Orange** - Highlighted information or intermediate steps
- **Red** - Errors or warnings
- **Purple** - Additional runs in complex examples
- **Gray** - Pencil marks or candidate digits
