# Master Kakuro Website - Design Guide

## Visual Theme

The website uses a **zen/natural aesthetic** inspired by the book cover, creating a calming, sophisticated brand experience.

## Color System

### Primary Colors
- **Sage Green** (#8B9D83) - Main background, conveys calm and nature
- **Cream** (#F5E6D3) - Accent color for CTAs and highlights, warm and inviting

### Secondary Colors
- **Sage Light** (#A8B8A0) - Lighter backgrounds for sections
- **Sage Dark** (#6B7D63) - Darker accents and borders
- **Wood** (#D4B896) - Warm secondary accent
- **Bamboo** (#7A9A65) - Vibrant accent for highlights
- **Stone** (#8A8A8A) - Neutral gray for subtle elements
- **Charcoal** (#2D3436) - Primary text color

## Typography

- **Font Family**: System sans-serif (clean, modern, accessible)
- **Headings**: Bold, large sizes (3xl-5xl) for visual hierarchy
- **Body Text**: Regular weight, 16-18px for readability
- **Line Height**: 1.5 for comfortable reading

## Component Styles

### Buttons
- **Primary CTA**: Cream background with charcoal text
- **Hover State**: Darker cream (#E8D4BD) with scale transform
- **Padding**: 16-20px horizontal, 12-16px vertical
- **Border Radius**: 8px for modern feel

### Cards
- **Background**: White/cream with subtle transparency
- **Shadow**: Soft shadow for depth
- **Padding**: 24px
- **Border Radius**: 12px

### Sections
- **Spacing**: 80px vertical padding (20px on mobile)
- **Max Width**: 1024px for content
- **Horizontal Padding**: 16px on mobile, 32px on desktop

## Responsive Breakpoints

- **Mobile**: 320px - 768px (single column)
- **Tablet**: 768px - 1024px (2 columns)
- **Desktop**: 1024px+ (full layout)

## Accessibility

- Minimum touch target size: 44px
- Color contrast: WCAG AA compliant
- Semantic HTML structure
- Alt text for all images
- Focus states for keyboard navigation

## Animation & Interaction

- **Hover Effects**: Subtle scale (1.05) and color transitions
- **Transitions**: 300ms duration for smooth feel
- **Scroll Behavior**: Smooth scrolling enabled

## Content Hierarchy

1. **Hero Section** - Largest, most prominent
2. **Benefits** - Secondary importance
3. **Features** - Detailed information
4. **Difficulty Levels** - Comparative data
5. **Final CTA** - Reinforcement
6. **Footer** - Minimal branding

## Image Optimization

- Book cover: 868 KB (optimized JPEG)
- Responsive sizing: 100% width, max-width 448px
- Hover transform for interactivity
- Rounded corners for modern aesthetic

## SEO Elements

- Semantic HTML (h1, h2, sections)
- Meta description and keywords
- Open Graph tags for social sharing
- Descriptive alt text
- Structured content hierarchy
